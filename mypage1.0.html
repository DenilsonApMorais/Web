  1. Criar arquivo padrão

    2. Para criar um arquivo padrão em html, criamos um novo arquivo atravéz do VSCODE e inicializamos com "!" e enter para completar o modelo padrão.

    Em seguida mudamos o idioma para pt-br e configuramos o title da page na tag head;


  2. Criar Títulos

    3. Para títulos e subtitulos podemos usar as tags h1 até h6;

    < h1>Título< /h1>

    < h2>Subtítulo< /h2>


  3. Links externos e internos

    • Link externo

      • Para configurar links usamos a tag "a". O primeiro parametro é o link desejado, o segundo target = "_blank" é para abrir o link em outra aba, rel= "external" é para informar ao mecanismo de buscas que é um link externo e não pertence ao meu conjunto de paginas.

      Por exemplo: < a href="https://www.youtube.com/" target="_blank" rel="external">YouTube.com < /a>

      Por exemplo: YouTube.com

    • Link interno

      • Podemos notar que a pagina desejada encontra-se na mesma pasta que o arquivo HTML;

      neste caso usamos rel ="next" para indicar que esta é a proxima pagina depois do meu index.html

      Por exemplo:< a href="imagens.html" rel="next">Pagina de Imagens< /a>

      Por exemplo:Pagina de Imagens


    • Link interno em pasta diferente do arquivo.

      • Também podemos ligar a paginas que estão em outras pastas usando: nomepasta/nomearquivo.html

      neste caso usamos rel ="next" para indicar que esta é a proxima pagina depois do meu index.html

      Por exemplo:< a href="exemplo/exemplo.html">Link de exemplo< /a>

      Por exemplo:Link de exemplo


    • Buscador não acessa esste link presente presente na pagina.

      • Podemos também garantir que o buscador não acesse um link presente em nossas paginas de terceiros.

      Usando rel = "nofollow"

      Por exemplo: < a href="imagens.html" rel="nofollow">Pagina de imagens< /a>

      Por exemplo:Pagina de imagens

    • Link para donwloads

      • Para fazer donwloads

      Neste caso clicamos em cima do link com a tecla direita e salvar link como..

      Por exemplo: < a href="exemplo/meu livro.pdf"download = "meulivro.pdf" type="aplication/pdf" >My Book< /a>

      Link do livro em PDF: My Book

      Posteriormente é só mudar o type para o tipo de arquivo desejado.

      Por exemplo: < a href="exemplo/meu livro.zip" download="meu livro.zip" type="aplication/zip">My Book< /a>

      Link do livro em ZIP: My Book


  4. Imagens

    5. para imagens usamos a tag < img src="" alt=""> como podemos ver essa tag fecha sozinha.

    • Imagens da web

      • < img src="https://st2.depositphotos.com/4164031/6914/i/450/depositphotos_69145633-stock-photo-flag-of-brazil.jpg" alt="esta é uma imagem de um olho com a bandeira do brasil">

      O primeiro parametro é o link da imagem, e o segundo é a tag alternativa, que sera mostrado caso a imagem não seja carregada, ou caso algum leitor de tela esteja sendo usado.

    • Imagens locais

      • O primeiro parametro é o caminho de onde está a imagem (Pode estar na mesma pasta ou não), e a tag alt é caso a imagem não seja carregada.

      < img src="imagens/urus.jpg" alt="Lamborghini Urus">

  5. Listas

    6. Listas podem ser ordenadas, não ordenadas, ou de definição

    Lista ordenada

                  
                < ol type="square" >
                    < li>Item da lista ordenada.< /li>
                < /ol>

    Lista não ordenada.

                
                < ul type ="circle">
                    < li>Item da lista não ordenada< /li>
                < /ul>

    Lista de definição.

                
                < dl>
                    < dt>Termo< /dt>
                    < dd>Descrição do termo< /dd>
                < /dl>
